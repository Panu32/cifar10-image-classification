# 🖼️ CIFAR-10 Image Classification using CNN

## 📌 Overview

This project implements a Convolutional Neural Network (CNN) to classify images from the CIFAR-10 dataset into 10 different categories such as airplanes, automobiles, birds, cats, deer, dogs, frogs, horses, ships, and trucks.

The model is trained using data augmentation techniques and evaluated using multiple performance metrics including accuracy, precision, recall, and confusion matrix visualization.

---

## 🚀 Features

* ✅ Custom CNN architecture built from scratch
* ✅ Data augmentation (image shifting & flipping)
* ✅ Performance evaluation using:

  * Accuracy
  * Precision & Recall
  * Confusion Matrix
* ✅ Visualization of training vs validation metrics
* ✅ Prediction visualization with confidence scores
* ✅ Analysis of correct and incorrect predictions

---

## 🧠 Model Architecture

The CNN model consists of:

* 3 Convolutional blocks (Conv2D + BatchNorm + MaxPooling + Dropout)
* Fully connected Dense layers
* Softmax output layer for multi-class classification
